# Repo Description
Repo for solving the follow-up problem by Lambda Capital


## Follow-up Problem
El problema consiste en desarrollar un "backtest" para una estrategia que opera en empresas dentro del S&P 500 del 2010-01-01 al 2020-01-01. Cada día la estrategia selecciona las 10 acciones con mayor ROC y las compra de manera equitativa.

No olvidar verificar que sólo seleccione entre acciones que son parte del S&P 500 en ese momento del tiempo, ya que cambian cada trimestre. 

Se debe de generar un archivo con los valores del equity curve (Net Asset Value) a través del tiempo y un archivo de pesos o posiciones donde se puedan ver las acciones que se seleccionaron cada día.
